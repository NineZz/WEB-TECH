My Work

Motion Detector with App Notification

 Motion Detector with App Notification สร้างขึ้นมาเพื่อใช้ในการตรวจจับการเคลื่อนไหว โดยใช้ IR Infrared Obstacle Avoidance Sensor Module ซึ่งเป็นโมดูลเซ็นเซอร์แสงสำหรับตรวจจับสิ่งกีดขวาง โดยโมดูลนี้ จะมีตัวรับและตัวส่ง infrared ในตัว ตัวสัญญาณ infrared จะส่งสัญญาณออกมา และเมื่อมีวัตถุมาบังคลื่นสัญญาณ infrared ที่ถูกส่งออกมาจะสะท้อนกลับไปเข้าตัวรับสัญญาณ สามารถนำมาใช้ตรวจจับวัตถุที่อยู่ตรงหน้าได้ และสามารถปรับความไว ระยะการตรวจจับ ใกล้หรือไกลได้ โดยหากมีวัตถุเคลื่อนที่ผ่าน จะใช้สัญญาณ WiFi ในการส่งการแจ้งเตือนผ่าน Line เพื่อแจ้งเตือนให้ผู้ใช้งานได้ทราบ

SpaceShip-Simulator

 SpaceShip-Simulator ถูกสร้างขึ้นมาเพื่อให้ผู้เล่นได้ผ่อนคลาย และสนุกไปกับเกม หรืออาจใช้เป็นกิจกรรมในยามว่างก็ได้ โดยเกมนี้ถูกสร้างขึ้นด้วยภาษา Python โดยใช้ Library ที่มีชื่อว่า Pygame วิธีการเล่นนั้นก็แสนจะง่ายดาย เราสามารถเริ่มเกมด้วยการกดคลิ้กหนึ่งครั้ง แล้วจากนั้นยานของศัตรูจะเริ่ม generate ออกมา และเพิ่มขึ้นเรื่อยๆตามด่านที่ผ่านไป ซึ่งเงื่อนไขของการแพ้ คือ ยานของศัตรูสามารถเข้าไปถึงด้านล่างของหน้าจอได้ 10 ครั้ง(พลาดได้ 10 ครั้ง) หรือจนกว่าเราจะโดนศัตรูยิงจนพลังชีวิตหมด

Project Multimedia
